## Problem Statement
An e-commerce company requires a comprehensive analytics solution to understand business performance and customer behavior. This project delivers a Tableau dashboard integrating sales, customers, products, payments, returns, and delivery data to support strategic decision-making.

## Objectives
- Analyze sales, revenue, and profit trends
- Segment customers using RFM analysis
- Identify top-performing and low-performing products
- Evaluate delivery performance, cancellations, and returns
- Study payment preferences and refund trends

## Dataset Overview
- Orders, customers, products, and sales data
- Payment methods and transaction details
- Returns and refund reasons
- Delivery timelines and shipping data

## Tools & Technologies
- Tableau
- Calculated Fields and Parameters
- Excel / CSV
- Data Analysis and Visualization

## Key Insights
- Identified high-value, loyal, and churn-risk customers
- Highlighted top-selling products and categories
- Analyzed delivery delays and operational bottlenecks
- Discovered return and refund trends affecting profitability
